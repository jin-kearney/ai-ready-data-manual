# 별첨 C — 유즈케이스 레이어 설계 기획서 (빈 템플릿)

> 한 줄: **9단계 중 8단계(유즈케이스 레이어)의 산출물 양식.** **코어를 재정의하지 않는다** — 코어([별첨 B](B-3%20별첨%20B%20—%20코어%20온톨로지%20설계%20기획서.md)) 위에서 특정 목적의 유즈케이스를 실행할 전용 구조만 설계하고, 코어에 이미 있는 것은 **참조**로만 연결한다.

> **쓰는 법:** 메인 [§6.3 유즈케이스 설계](../B-3%20온톨로지.md#63-유즈케이스-레이어-설계-8단계--산출물-유즈케이스-기획서)를 읽으며 채운다. 목적 유형별 설계 차이·**작성 사례(CCL)**는 [별첨 A 각론](B-3%20별첨%20A%20—%20유즈케이스%20온톨로지%20구축%20각론.md).
