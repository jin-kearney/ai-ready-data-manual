## 핵심 내용

### 정의
엔터프라이즈 데이터 자산 중 데이터 카탈로그에 색인되고 문서화된 비율.
데이터의 조직화 정도와 검색 가능성을 나타냄.

### 측정 공식
**문서화된 데이터 자산 수 / 전체 데이터 자산 수**

### 목표 수치(%)
| 범위 | 평가 |
|------|------|
| **80% 이상** | 우수한 데이터 거버넌스 및 접근성 |
| 60~79% | 중간 수준 커버리지; 개선 필요 |
| 60% 미만 | 심각한 격차; 즉시 조치 필요 |

조직은 포괄적인 데이터 활용을 보장하기 위해 최소 80% 이상의 커버리지를 목표로 해야 함.
