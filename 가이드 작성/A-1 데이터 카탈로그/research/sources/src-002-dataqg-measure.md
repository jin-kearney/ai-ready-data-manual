## 핵심 내용

데이터 카탈로그 플랫폼 성공을 측정하는 4개 영역 KPI 프레임워크.

### Measure 1: 사용자 가치 제공 (Provides value to end-users)
1. **총 사용자 온보딩 수**: 플랫폼 도입 사용자 누적 수
2. **주/월간 신규 사용자 수**: 도입 속도 지표
3. **일일 검색 쿼리 수**: 검색이 데이터 카탈로그의 핵심 기능이므로 검색 활용도 측정
4. **데이터 자산 페이지 조회수**: 월별 조회수 추이

### Measure 2: 개방형 생태계 (Provides open eco-system)
1. **신규 데이터 소스 온보딩 수**: 연결된 데이터 소스 수
2. **연동 애플리케이션 수**: 통합 확장성
3. **기능 배포 사이클 타임**: 신기능 개발~배포 소요 기간

### Measure 3: 조직 가치 추가 (Adds value to the organization)
1. **일일 활성 사용자(DAU)**: "The more active users show users have built trust in the platform"
2. **고유 데이터 자산 페이지 조회 수**: 다양한 자산 활용도

### Measure 4: 신기능 도입 활성화 (Improves Adoption with New features)
1. **신기능 트래픽**: 신기능 페이지 방문 빈도와 체류율
2. **신기능으로 인한 신규 사용자 온보딩**: 기능 기반 확장성
