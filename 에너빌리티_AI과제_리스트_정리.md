### 20개 주제 밖에서 새로 나온 것 (주제 체계로 흡수되지 않는 테마)
1. **AI 운영모델·거버넌스** — AI 과제 실패율 90% → 외주 의존 탈피·현업 주도 'AI Native' Sandbox 개발. 또한 "온톨로지 데이터를 만드는 부서와 시스템 수혜 부서가 별개라 전사 워크플로우 정의 필요"(정병일 팀장). → 데이터 주제가 아닌 **조직·운영모델** 영역.
2. **AI 인프라·플랫폼** — GPU 서버 구입, MCP Tools 플랫폼 구축, 망분리 SDN, HPC 라이선스 최적화. → 데이터 체계가 아닌 **인프라** 영역(과제 리스트엔 다수 존재).
3. **기술 선택 판단 기준** — "온톨로지 vs RAG vs RDB/Text-to-SQL을 언제 쓸지", "99.99%가 아닌 100% 정확이 필요한 업무엔 온톨로지" 같은 **하이브리드 아키텍처 의사결정 가이드**. B-3에 일부 걸치나, '기술 선택 메타 가이드'는 현 주제 정의 범위를 다소 넘어섬 → B-3 적용 판단 기준 보강 시 흡수 가능.
