# Securiti.ai — AI 데이터 보안 솔루션

## 기본 정보

| 항목 | 내용 |
|---|---|
| 개발사 | Securiti (실리콘밸리; 2019 설립) — 2025.12 Veeam Software에 $1.725B에 인수 |
| 라이선스 | 상용 (Veeam 인수 후 Securiti Data Command Center는 독립 제품으로 운영 유지) |
| 배포 형태 | SaaS / Hybrid (프라이빗 클라우드·온프레미스 연결 지원) |
| 최신 동향 | Veeam 인수 완료 (2025.12) — 데이터 복원력(백업)+DSPM+AI 거버넌스 통합 "Data Command Center" 비전; LLM Firewall 출시; 100+ 글로벌 규제 지원; OWASP Top 10 LLM 리스크 대응 플랫폼으로 포지셔닝 |

## 한 줄 포지셔닝

> **DSPM·AI 거버넌스·프라이버시 자동화를 통합한 Data Command Center**. 100+ 글로벌 개인정보 규제를 단일 플랫폼으로 관리하고, LLM Firewall로 GenAI 애플리케이션의 OWASP Top 10 AI 리스크를 방어하는 가장 포괄적인 AI 데이터 보안 플랫폼이다.

---

## 주요 기능

### Data Command Center (핵심 플랫폼)
- **하이브리드 멀티클라우드 통합**: AWS, Azure, GCP, 온프레미스 데이터 소스를 단일 콘솔로 통합 가시화·제어
- **자동 데이터 탐지·분류**: 정형 DB, 비정형 문서(PDF, Word, 이메일), 이미지, 오디오에서 PII·PHI·금융 데이터 탐지. NLP+ML 기반 정밀 분류
- **DSPM(데이터 보안 포지션 관리)**: 전체 데이터 자산의 보안 취약점·과노출·규제 위반을 실시간 위험 점수로 제공
- **데이터 리니지**: 데이터 이동 경로·변환 이력 추적. 규제 감사 증빙

### LLM Firewall (GenAI 보안 — 핵심 차별점)
- **컨텍스트 인식 Prompt 분석**: 단순 키워드가 아닌 문맥 기반 PII·기밀 탐지. 다국어 지원
- **OWASP Top 10 LLM 리스크 방어**:
  - Prompt Injection 탐지·차단
  - 민감 정보 노출 방지
  - 적대적 공격(Adversarial Attacks) 탐지
  - 유해 콘텐츠 필터링
- **멀티미디어 콘텐츠 검사**: 텍스트·이미지·문서가 포함된 멀티모달 Prompt 분석
- **NIST AI RMF·EU AI Act 준수**: LLM 운영 위험 관리 프레임워크 자동 적용

### 프라이버시 자동화 (PrivacyOps)
- **100+ 규제 매핑**: GDPR, CCPA/CPRA, LGPD, PIPEDA, PIPL(중국), 한국 개인정보보호법(PIPA), EU AI Act 등 자동 매핑·컴플라이언스 갭 분석
- **DSAR 자동화**: 정보주체 열람·삭제·이동 요청 처리 워크플로 자동화
- **동의 관리**: 처리 동의 기록·추적·만료 관리
- **데이터 최소화**: 보존 기간 초과 데이터 자동 식별·삭제 권고

### AI 거버넌스
- **AI 시스템 인벤토리**: 조직 내 모든 AI 모델·데이터셋·AI 도구 자동 탐지·목록화
- **AI 위험 평가**: EU AI Act 준수를 위한 AI 시스템 위험 분류(금지·고위험·저위험)
- **Shadow AI 모니터링**: 미승인 AI 도구 사용 탐지·정책 적용
- **Veeam 통합(2026 예정)**: 데이터 백업·복원 + 보안 포지션 통합 — 랜섬웨어 공격 후 안전한 데이터 복구 시 민감 데이터 무결성 확인

## 강점

- **가장 포괄적인 AI 데이터 보안**: DSPM + LLM Firewall + AI 거버넌스 + 프라이버시 자동화 단일 플랫폼 — 경쟁사 중 가장 넓은 AI 보안 커버리지
- **100+ 규제 지원**: 한국 PIPA를 포함한 글로벌 규제 자동 매핑. 다국가 운영 대기업에 최적
- **LLM Firewall 차별화**: OWASP Top 10 LLM 리스크 기반 방어 체계 — GenAI 서비스 도입 시 즉각 적용 가능
- **Veeam 통합 시너지**: 데이터 복원력(백업)과 보안의 통합 → 사이버 공격 복구 시 민감 데이터 무결성 보장 (2026~)

---

## 약점·주의점

- **Veeam 인수 전환기**: 제품 로드맵·가격 정책·지원 체계가 Veeam 통합 과정에서 변동 가능. 단기적 불확실성 존재
- **비용**: 모듈이 많아 전체 플랫폼 도입 시 상당한 비용. 필요 모듈만 선택적 도입 고려
- **세분화 접근 제어 한계**: Immuta, Privacera 수준의 데이터 플랫폼 네이티브 세분화 접근 정책(ABAC)은 상대적으로 약함
- **온프레미스 완전 배포**: 완전 온프레미스 배포는 Hybrid 모드로 가능하지만 SaaS 대비 기능 일부 제한
