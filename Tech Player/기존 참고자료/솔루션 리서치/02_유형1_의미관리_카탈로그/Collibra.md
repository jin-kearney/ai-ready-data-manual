## 주요 기능

### 1. 데이터 카탈로그 (2026-03 GA 업데이트)
- **자산 발견·등록**: 100+ 커넥터로 자동 메타데이터 수집
- **AI 자동 문서화**: LLM 기반 테이블·컬럼·다이어그램 설명 자동 생성
- **데이터 제품 관리**: 팀·도메인 간 데이터 제품 가시성 향상, 일관된 데이터 제품 전달
- **비정형 데이터 카탈로그 (2025 신기능)**: Deasy Labs 인수로 문서·전사·프레젠테이션 등 비정형 데이터 분류·거버넌스 확장

### 2. Business Glossary & 데이터 사전
- **계층적 Glossary**: 도메인→카테고리→용어의 계층 구조로 전사 표준 용어 관리
- **승인 워크플로우**: 용어 정의 제안→검토→승인→발행 프로세스 자동화
- **용어-자산 연결**: Glossary 용어를 카탈로그 내 데이터 자산(테이블·컬럼)에 태그로 연결
- **폴리시 연동**: Glossary 용어에 데이터 정책(접근·보호) 자동 적용

### 3. 데이터 리니지
- **크로스 플랫폼 자동 리니지**: 100+ 소스에서 ETL·SQL·BI 리니지 자동 수집
- **컬럼 레벨 리니지**: 소스 컬럼에서 대시보드 필드까지 완전한 리니지 추적
- **Impact Analysis**: 상위 자산 변경이 하위 리포트에 미치는 영향 즉시 파악
- **AI 리니지**: ML 모델 학습 데이터 소스부터 배포 결과까지 추적

### 4. 데이터 품질 (통합 DQ+Observability)
- **DQ 관리**: 품질 규칙 정의, 자동 체크 실행, 품질 스코어카드
- **데이터 관찰성(Observability) 통합 (2025 신기능)**: DQ와 Observability를 AI 거버넌스와 통합 — 신뢰 데이터 파이프라인 구현
- **품질 문제 워크플로우**: 품질 이슈 발견→담당자 알림→해결 추적

### 5. AI 거버넌스 (2025~2026 핵심 차별점)
- **AI 사용 사례 카탈로그**: AWS Bedrock, SageMaker, Azure AI Foundry, Azure ML, Databricks Unity Catalog, Google Vertex AI, MLflow, SAP AI Core 등 멀티클라우드 AI 자산 중앙 가시성
- **EU AI Act 준수 템플릿**: 위험 평가, 모델 문서화, 감사 증적 관리 자동화
- **NIST AI RMF 매핑**: AI 리스크 관리 프레임워크 준수 지원
- **ISO 42001 인증 (2025)**: AI 관리 시스템 국제 표준 인증 획득

### 6. 접근 제어 & 정책 관리
- **정책 중앙화**: 데이터 접근·보호·사용 정책을 Collibra에서 정의, 실행 플랫폼에 전파
- **TrustLogix 파트너십**: Snowflake 등 플랫폼과 정책 자동 동기화
- **역할 기반 거버넌스**: Creator/Contributor/Consumer 역할 구분, Consumer 무료

## 강점

1. **엔터프라이즈 거버넌스 최강**: 워크플로우·정책·AI 거버넌스를 가장 포괄적으로 제공하는 검증된 솔루션
2. **AI 거버넌스 선도**: EU AI Act·ISO 42001 대응, 멀티클라우드 AI 자산 중앙 관리
3. **규정 준수 자동화**: GDPR, CCPA, HIPAA 등 규제 준수를 워크플로우로 자동화
4. **비정형 데이터 거버넌스**: Deasy Labs 인수로 문서·전사 등 비정형까지 거버넌스 확장
5. **Gartner 장기 리더**: 수년간 Gartner 리더 포지션 유지로 대기업 조달 신뢰도 높음

## 약점·주의점

1. **높은 비용**: 기본 $170K+/년, 모듈(AI 거버넌스 $104K, DQ $156K+) 별도 과금 → 총 TCO 수억 원 수준
2. **구현 기간**: 구현에 수개월~1년 이상 소요, 전문 파트너 필수
3. **커스터마이징 복잡**: 기본 메타모델 외 커스텀 자산 타입 추가는 전문 지식 필요
4. **MCP 미지원**: AI 에이전트 통합 표준 프로토콜(MCP) 공식 지원 없음 (2026 기준)
5. **UX 개선 필요**: 비즈니스 사용자 채택률이 복잡한 UI로 인해 낮은 경우 많음
