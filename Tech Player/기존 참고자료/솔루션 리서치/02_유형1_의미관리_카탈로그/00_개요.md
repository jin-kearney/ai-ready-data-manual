# 유형① 의미 관리 — 전문 데이터 카탈로그/거버넌스 구분 B 개요

> 작성일: 2026-06-10 | 조사 기준: 2025~2026년 최신 릴리스 기준

---

## 1. 구분 정의

**전문 데이터 카탈로그/거버넌스 솔루션**은 통합 데이터 플랫폼(Lv.1)에 내장된 카탈로그를 보완하거나 대체하는, 의미·거버넌스 관리에 특화된 독립 솔루션이다.  
여러 데이터 플랫폼(Databricks, Snowflake, SAP, Oracle 등)을 동시에 운영하는 환경에서 **플랫폼 중립적으로** 전사 데이터 자산의 의미(Semantics)·신뢰(Trust)·거버넌스(Governance)를 중앙 관리한다.

### 커버하는 AI-Ready Data 주제

| 주제코드 | 주제명 | 커버 방식 |
|---------|--------|-----------|
| A-1 | 데이터 카탈로그 | 전문 카탈로그로 전사 자산 통합 발견·탐색 |
| A-2 | 메타데이터 관리 | 기술·비즈니스·운영 메타데이터 통합 관리 |
| A-3 | 데이터 리니지 | 크로스 플랫폼 엔드투엔드 리니지 |
| C-3 | Business Glossary | 전사 공통 비즈니스 용어 정의·승인·전파 |

**참고**: 유형① 솔루션은 접근 제어(C-1)·품질(E-1)·DataOps(F-1) 기능도 일부 포함하지만 Lv.1 플랫폼 대비 기능이 제한적이며, 별도 연동 구성이 필요하다.

### 5-2. 두산그룹 같은 다계열사 대기업 환경 관점

**핵심 고려사항:**

1. **멀티 플랫폼 환경**: 계열사마다 Databricks·Snowflake·Oracle·SAP·MySQL 등 다양한 데이터 소스가 혼재 → **플랫폼 중립 카탈로그 필수**. Collibra·Informatica·Atlan이 가장 폭넓은 커넥터 지원.

2. **거버넌스 중앙화**: 그룹 차원 데이터 표준(용어 사전, 분류 체계)을 계열사에 내려주는 **Top-Down 거버넌스**에는 Collibra의 워크플로우 엔진이 강점. 계열사 자율이라면 Atlan의 도메인 기반 분산 거버넌스가 유연.

3. **SAP 연동**: 두산그룹처럼 SAP ERP 기반 조직은 Informatica CDGC가 SAP 메타데이터 수집·연동에 유리 (Informatica SAP 커넥터 성숙도 높음).

4. **예산 대 효과**: Collibra·Informatica는 초기 투자가 크고 구현 기간 길지만 대규모 조직에 검증됨. Atlan은 빠른 온보딩·SaaS 편의성이 강점이나 커스텀 워크플로우는 제한.

5. **오픈소스 전략**: 예산 제약 계열사에는 DataHub/OpenMetadata 자체 호스팅으로 시작하고 그룹 중앙 솔루션(Collibra 등)과 연동하는 하이브리드 접근도 가능.

6. **AI 준비도**: 2026년 기준 AI 에이전트 통합을 최우선으로 한다면 Atlan(MCP 네이티브)과 DataHub(MCP 지원)가 앞서있음. Collibra와 Informatica는 AI 기능을 급속도로 추가 중이나 아직 에이전트 통합 성숙도는 낮음.
