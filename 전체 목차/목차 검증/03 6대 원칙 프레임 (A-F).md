# AI-Ready Data — 6대 원칙 프레임 (A~F)

> 20개 주제(A-1~F-4)는 데이터가 AI까지 도달하기 위해 갖춰야 할 **6대 원칙(상태)**으로 묶인다.
> FAIR 데이터 원칙(Findable·Accessible·Interoperable·Reusable)을 AI-Ready 관점으로 확장한 프레이밍이다.
> 한 줄 요약: **찾을 수 있고(A) → 이해할 수 있고(B) → 믿을 수 있고(C) → 실행에 쓸 수 있고(D) → 지속·순환되며(E) → 통제·기반이 갖춰진(F)** 데이터.

## 한눈에 보기

|------|------|-----------|-----------|-----------|
| **A** | **Findable** (발견·의미) | 데이터를 **찾을 수 있게** | "어디에 무엇이 있고, 무슨 뜻인가?" | A-1 카탈로그 · A-2 메타데이터 · A-3 Glossary |
| **B** | **Understandable** (변환·지식화) | AI가 **이해할 수 있게** | "AI가 읽고 맥락을 알 수 있나?" | B-1 전처리 · B-2 주석 · B-3 온톨로지 |
| **C** | **Trustworthy** (신뢰·관측) | 믿고 **쓸 수 있게** | "이 데이터를 믿고 써도 되나?" | C-1 Observability · C-2 품질 · C-3 Lineage |

## A. Findable — 데이터를 찾을 수 있게
**원칙:** AI와 사용자가 필요한 데이터 자산의 소재·의미를 즉시 찾을 수 있어야 한다. 못 찾는 데이터는 없는 데이터다.
- **A-1 데이터 카탈로그** — 자산의 소재·오너·접근 경로 등록 ("어디에 있나")
- **A-2 메타데이터** — 자산의 구조·단위·기준 속성 기술 ("어떤 속성인가")
- **A-3 비즈니스 Glossary** — 용어·약어 표준 정의 ("그 말이 무슨 뜻인가")

## B. Understandable — AI가 이해할 수 있게
**원칙:** 찾은 데이터를 AI가 읽고 의미·맥락까지 해석할 수 있어야 한다.
- **B-1 데이터 전처리** — 비정형/반정형을 AI가 읽을 구조로 변환
- **B-2 데이터 해설/주석** — 학습용 라벨·해석 기준 부여
- **B-3 온톨로지** — 개념 간 관계·인과 구조로 맥락 부여

## C. Trustworthy — 믿고 쓸 수 있게
**원칙:** AI에 투입되는 데이터가 신뢰할 수 있고, 결과의 근거를 추적할 수 있어야 한다.
- **C-1 데이터 Observability** — 운영 중 이상 징후 실시간 감지·알림
- **C-2 데이터 품질 관리** — "쓸 수 있는가" 판정 게이트 + 접근 권한
- **C-3 데이터 계통 Lineage** — 출처→변환→AI 답변 근거 추적

## D. Actionable — 실행에 쓸 수 있게
**원칙:** 데이터가 실제 현장·시스템·Agent의 행동으로 연결될 수 있어야 한다.
- **D-1 Physical 데이터** — 센서·설비 신호를 AI 입력으로 연결·표준화
- **D-2 API/Tool 연계 데이터** — Agent가 외부 시스템을 안전 호출하는 명세
- **D-3 Prompt/Harness 자산화** — 업무 절차·판단을 AI 실행 자산으로

## E. Sustainable — 지속·순환되게
**원칙:** 한 번 쓰고 끝이 아니라 재사용·검증·환류로 계속 좋아져야 한다.
- **E-1 데이터 Product화** — 재사용 가능한 상품 단위로 책임지고 제공
- **E-2 합성데이터** — 부족·제한 데이터를 인공 생성으로 보충
- **E-3 AI 평가 데이터** — 성능을 객관 판단하는 정답셋·기준
- **E-4 데이터 Feedback Loop** — 운영 결과·오류를 개선으로 환류

## F. Governed — 통제·기반이 갖춰지게
**원칙:** 위 모든 것이 안정·안전·정책의 기반 위에서 굴러가야 한다.
- **F-1 데이터 운영관리 (DataOps)** — 파이프라인 안정 실행·복구·변경 관리
- **F-2 데이터 생애주기 관리** — 생성~폐기 시간축 정책 통제
- **F-3 데이터 디지털화** — 아날로그·암묵지를 디지털 원천으로 전환
- **F-4 AI 데이터 보안** — 민감정보 탐지·마스킹·비식별

---

> 본 프레임은 기존 한국어 가족명(발견·의미 / 변환·지식화 / 신뢰·관측 / 활용·실행 / 가치화·순환 / 운영·기반)과 1:1 대응한다. 장표 표지·간지 등에 영문 원칙명(Findable~Governed)을 함께 노출하면 "데이터가 AI까지 가는 6단계"를 한 줄로 전달할 수 있다.
